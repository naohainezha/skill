========================================
  小红书爬虫 - V4.0 已发布
========================================

【最新推荐：V4.0】

基于成功的 xhs-crawler 项目
采用"简单但真实"的策略

【核心改进】

✅ 同步 Playwright - 更稳定
✅ 精准等待 - wait_for_selector
✅ 智能滚动 - 检测页面高度
✅ 完善调试 - 截图+HTML
✅ 真实环境 - 简单但真实

【性能对比】

V3.0: 15-25篇/分, 成功率 90-95%
V4.0: 10-15篇/分, 成功率 95-99%

【快速开始】

Windows: run_v4.bat
Linux/Mac: python crawler_v4.py

【使用步骤】

1. 运行上述命令
2. 扫码登录
3. 等待采集完成（7-10分钟）
4. 查看 output 目录

【配置文件】

编辑 .env 文件：
  KEYWORDS=眼镜
  TARGET_COUNT=100
  SLEEP_BETWEEN_REQUESTS=3.5
