# 第六步：质量自检

## 你的任务

在输出最终结果前，逐项检查以下内容。

## 自检清单

### 标题检查
- [ ] 字数在15-20字之间？
- [ ] 包含目标人群词（脸型/人群）？
- [ ] 包含热搜关键词（材质/款式）？
- [ ] 没有使用禁止词汇？

### 正文检查
- [ ] 字数在120-180字之间？
- [ ] 使用第一人称真诚分享？
- [ ] 包含真实优缺点（不只有好）？
- [ ] 避开所有禁止词汇？
- [ ] 语气真实，没有营销痕迹？

### 标签检查
- [ ] 标签数量在3-10个之间？
- [ ] 必包含 #眼镜？
- [ ] 有脸型标签？
- [ ] 有场景标签？
- [ ] 没有重复或无关标签？

### 总体检查
- [ ] 总字数（标题+正文+标签）在200字以内？
- [ ] 符合 rules/禁止清单.md 要求？

## 完成标志

1. 显示优化后的完整笔记
