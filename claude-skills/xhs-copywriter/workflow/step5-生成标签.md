# 第五步：生成标签

## 你的任务

生成3-10个精准标签。

## 必需标签（3个）

### 1. 核心标签
- `#眼镜` （必须）

### 2. 脸型标签（选1个）
- `#方圆脸` - 方圆脸适用
- `#菱形脸` - 颧骨高适用
- `#圆脸` - 圆脸适用

### 3. 场景标签（选1-2个）
- `#打工人` - 上班族
- `#学生党` - 学生群体
- `#早八人` - 早起通勤
- `#通勤` - 日常使用

## 建议标签（可选）

根据内容选择添加：
- `#高度近视` - 如果提到度数
- `#金丝眼镜` / `#银丝眼镜` - 材质
- `#纯钛` - 材质
- `#无框眼镜` - 款式

## 标签规则

- [ ] 总数量：3-10个
- [ ] 格式：每个标签前加#，用空格分隔
- [ ] 避免重复：不要用意思相近的标签
- [ ] 精准优先：选择最相关的，而不是堆砌
