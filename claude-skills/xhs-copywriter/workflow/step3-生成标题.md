# 第三步：生成标题

## 你的任务

根据分析的关键信息，生成15-20字的标题。

## 标题公式

**核心结构**：[目标人群] + [热搜关键词]

### 示例模板

1. **痛点风**：脸型 + 避雷/推荐
   - "方圆脸避雷！粗黑框让我土了20年"

2. **反差风**：不戴vs戴
   - "不戴眼镜纯，戴眼镜欲"

3. **人设风**：人群 + 风格词
   - "打工人超轻银丝框，通勤戴着真香"

4. **场景风**：场景 + 随手试
   - "随手试了副纯钛框，意外很合适"

## 必需元素

- [ ] 包含脸型或人群词
- [ ] 包含材质/款式关键词
- [ ] 字数控制在15-20字
