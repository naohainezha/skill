# 第二步：分析待优化内容

## 你的任务

提取原文中的关键信息，包括：

### 1. 脸型信息
- 方圆脸、菱形脸、圆脸、长脸等
- 如果未明确，需询问用户确认

### 2. 眼镜信息
- 材质：纯钛、金丝、银丝、普通金属
- 款式：小方框、椭圆框、黑框、无框
- 特点：超轻、粗框、细框

### 3. 使用场景
- 通勤、上学、日常、工作
- 打工人、学生党、早八人

### 4. 其他信息
- 近视度数（如提到）
- 佩戴感受（如提到）
- 优缺点（如提到）
