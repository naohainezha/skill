# 小红书笔记模板

---

## 标准输出格式

```
[标题]

[正文]

[标签]
```

---

## 结构说明

### 1. 标题
- **位置**：第一行
- **字数**：15-20字
- **格式**：[目标人群] + [热搜关键词] + 可选emoji（👓）
- **示例**：方圆脸超轻银丝框，纯钛戴着真香👓

### 2. 正文
- **位置**：标题后空一行
- **字数**：120-180字
- **结构**：
  - 开场白（1-2句）
  - 使用感受：优点 + 缺点（2-3句）
  - 总结（1句）
- **语气**：第一人称真诚分享
- **示例**：
  ```
  今天试了副银丝小方框，纯钛材质的。超轻，戴一整天鼻梁都不疼，感觉重量真的可以忽略。银丝的质感很精致，戴上看起来斯文了不少，同事也说比之前的黑框显气质。不过银丝确实容易变形，放的时候要注意收纳。日常通勤戴刚好，视野也清晰，就是镜片有点厚，侧面看能看见折射的边缘。
  ```

### 3. 标签
- **位置**：正文后空一行
- **数量**：3-10个
- **格式**：#标签1 #标签2 #标签3（空格分隔）
- **必需**：
  - #眼镜（核心标签）
  - 脸型标签（#方圆脸/#菱形脸/#圆脸）
  - 场景标签（#打工人/#学生党/#早八人/#通勤）
- **示例**：#眼镜 #方圆脸 #打工人 #通勤 #银丝眼镜 #纯钛

## 完整示例

```
方圆脸超轻银丝框，纯钛戴着真香👓

今天试了副银丝小方框，纯钛材质的。超轻，戴一整天鼻梁都不疼，感觉重量真的可以忽略。银丝的质感很精致，戴上看起来斯文了不少，同事也说比之前的黑框显气质。不过银丝确实容易变形，放的时候要注意收纳。日常通勤戴刚好，视野也清晰，就是镜片有点厚，侧面看能看见折射的边缘。

#眼镜 #方圆脸 #打工人 #通勤 #银丝眼镜 #纯钛
```

---

## 使用说明

在 `workflow/step6-质量自检.md` 中，按照此模板生成最终输出：

1. 替换 `[标题]` 为实际生成的标题
2. 替换 `[正文]` 为实际生成的正文
3. 替换 `[标签]` 为实际生成的标签

确保所有占位符都已替换为实际内容。
