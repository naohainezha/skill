# 小红书眼镜文案写作指南

## 核心原则

### 真实感第一

所有文案必须像真人分享，包含真实的使用场景和感受：
- 第一人称叙述
- 包含优缺点
- 加入轻度吐槽
- 自然的情绪表达

### 避开雷区

**严禁使用的词汇：**
- 高智感、显脸小、焊死在脸上
- 香疯了、yyds、绝美、疯狂囤货
- 求链接、看主页、已下单、🔗

**禁止的行为：**
- 只说好不说坏
- 夸张表述
- 强营销引导
- 堆砌优点

## 标题公式

### 基础结构
```
[情绪钩子] + [眼镜动作] + [人设标签] + [场景] + 👓 + #核心标签
```

### 爆款模板

1. **痛点风**：方圆脸避雷！粗黑框眼镜让我土了20年👓
2. **反差风**：不戴眼镜纯，戴眼镜欲👓 这副金丝框我焊死了
3. **场景风**：在公司卫生间对镜拍👓 没人给拍照的可怜博主
4. **疑问风**：戴眼镜vs不戴眼镜，哪个更好看？👓

## 正文写作技巧

### 开场白（1-2句）
- 随手试了副...
- 今天配了新眼镜...
- 周末去试了几副框...

### 使用感受（2-3句）
- 必须包含优点：戴着舒服、好看、合适
- 必须包含缺点：有点重、视野盲区、容易变形

### 总结（1句）
- 最后选了XX，日常戴着刚好
- 姐妹们说看起来XX了不少

## 标签组合

### 必需标签（3个）
```
#眼镜 + 脸型标签 + 场景标签
```

### 脸型标签选择
- #方圆脸（方圆脸适用）
- #菱形脸（颧骨高适用）
- #圆脸（圆脸适用）

### 场景标签选择
- #打工人（上班族）
- #学生党（学生群体）
- #早八人（早起通勤）
- #通勤（日常使用）

## 字数控制

- 标题：15-20字
- 正文：120-180字
- 标签：3-10个
- **总计：不超过200字**

## 质量检查

发布前必须检查：

- [ ] 标题字数15-20字？
- [ ] 标题包含目标人群 + 热搜关键词？
- [ ] 正文第一人称真诚分享？
- [ ] 包含真实优缺点？
- [ ] 避开所有禁止词汇？
- [ ] 标签3-10个？
- [ ] 总字数200字以内？
