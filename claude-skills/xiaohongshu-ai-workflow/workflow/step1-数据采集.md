# 第一步：数据采集

## 你的任务

使用小红书爬虫采集相关笔记数据。

## 执行步骤

### 1. 切换到爬虫目录
```bash
cd D:\xiaohongshu-crawler
```

### 2. 运行爬虫脚本
```bash
python search_xhs.py "关键词" 数量
```

**示例**：
```bash
python search_xhs.py "方圆脸眼镜" 20
```

### 3. 确认输出文件
采集的数据会保存到：
```
D:\xiaohongshu-crawler\output\notes_*.json
```

文件命名格式：`notes_关键词_时间戳.json`

## 数据格式

输出的 JSON 文件包含以下字段：
- 作品标题
- 作者信息
- 发布时间
- 作品标签
- 互动数据（收藏、评论、点赞、分享数）
- 作品内容

## 完成标志

当成功采集到指定数量的笔记后，进入下一步。

## 注意事项

⚠️ **采集前确认**：
- 网络连接稳定
- 爬虫脚本可执行
- 关键词明确具体
