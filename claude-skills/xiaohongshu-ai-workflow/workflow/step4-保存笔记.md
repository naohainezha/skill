# 第四步：保存笔记

## 你的任务

将生成的笔记保存到指定位置。

## 输出位置

### JSON 格式
```
D:\xiaohongshu-crawler\output\kimi_generated_notes_关键词.json
```

### TXT 格式（可读版）
```
D:\xiaohongshu-crawler\output\kimi_generated_notes_关键词.txt
```

## 文件命名

文件命名格式：`kimi_generated_notes_关键词`

**示例**：
- `kimi_generated_notes_方圆脸眼镜.json`
- `kimi_generated_notes_方圆脸眼镜.txt`

## JSON 格式结构

```json
[
  {
    "标题": "笔记标题",
    "正文": "笔记正文",
    "标签": "#标签1 #标签2 #标签3"
  }
]
```

## TXT 格式结构

```
========================================
        笔记一：标题
========================================

正文

标签：#标签1 #标签2 #标签3

----------------------------------------

        笔记二：标题
========================================

正文

标签：#标签1 #标签2

========================================
```

**详细格式**：见 [templates/笔记输出格式.md](../templates/笔记输出格式.md)
