# 项目结构

```
C:\Users\admin\Projects\XHS-Downloader\
├── source\
│   ├── application\
│   │   └── app.py          # 核心应用类 XHS
│   ├── expansion\
│   │   └── browser.py       # 浏览器 Cookie 读取（可选）
│   ├── module\              # 数据提取、下载管理
│   └── translation\        # 国际化
├── Volume\                 # 默认工作目录
│   ├── Download\            # 下载文件保存位置
│   ├── ExploreData.db       # 作品数据数据库
│   ├── ExploreID.db         # 下载记录
│   └── settings.json       # 配置文件
├── main.py                # 程序入口
└── requirements.txt        # 依赖列表
```

## 目录说明

### source/
核心应用代码，包含：
- `application/app.py`：XHS 类，提供 extract() 方法
- `expansion/browser.py`：可选的浏览器 Cookie 读取功能
- `module/`：数据提取和下载管理模块
- `translation/`：国际化支持

### Volume/
工作目录，包含：
- `Download/`：下载的图片和视频文件
- `ExploreData.db`：作品元数据数据库
- `ExploreID.db`：下载记录，用于去重
- `settings.json`：用户配置文件

### 根目录
- `main.py`：程序入口
- `requirements.txt`：Python 依赖列表

## 数据库说明

### ExploreData.db
存储作品元数据，包括：
- 作品标题
- 作者信息
- 发布时间
- 标签
- 互动数据

### ExploreID.db
存储已下载的作品ID，用于：
- 自动跳过重复下载
- 提高下载效率
