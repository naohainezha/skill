# 支持的链接格式

## 完整作品链接

### 1. 探索页作品
```
https://www.xiaohongshu.com/explore/作品ID?xsec_token=XXX
```

### 2. 发现页作品
```
https://www.xiaohongshu.com/discovery/item/作品ID?xsec_token=XXX
```

### 3. 用户主页作品
```
https://www.xiaohongshu.com/user/profile/作者ID/作品ID?xsec_token=XXX
```

## 短链接

### xhslink 分享码
```
https://xhslink.com/分享码
```

程序会自动解析短链接，获取完整作品链接。

## 链接示例

### 有效链接示例
- `https://www.xiaohongshu.com/explore/6954ec6700000000220329a6`
- `https://www.xiaohongshu.com/discovery/item/6954ec6700000000220329a6`
- `https://www.xiaohongshu.com/user/profile/5c6fb3fb00000000110126a2/6954ec6700000000220329a6`
- `https://xhslink.com/abc123def456`

### 无效链接示例
- `https://xiaohongshu.com/explore/xxx` （缺少 www）
- `www.xiaohongshu.com/explore/xxx` （缺少 https://）
- `https://www.xiaohongshu.com/explore/` （缺少作品ID）

## 提取作品ID

从完整链接中提取作品ID：

```
https://www.xiaohongshu.com/explore/6954ec6700000000220329a6?xsec_token=xxx
                                     ↑ 作品ID ↑
```

作品ID格式：24位十六进制字符串。

## xsec_token

- **作用**：安全令牌，防止爬虫
- **影响**：部分链接需要 xsec_token 才能访问
- **处理**：程序会自动处理，无需手动提取

如链接无 xsec_token，程序仍会尝试访问。
