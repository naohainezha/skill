# 配置说明

## 配置文件位置

```
C:\Users\admin\Projects\XHS-Downloader\Volume\settings.json
```

## 主要参数

```json
{
  "work_path": "C:\\Users\\admin\\XHS-Downloader\\Volume",
  "folder_name": "Download",
  "name_format": "发布时间 作者昵称 作品标题",
  "image_format": "PNG",
  "image_download": true,
  "video_download": true,
  "download_record": true,
  "folder_mode": false,
  "author_archive": false
}
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `work_path` | 工作目录路径 | Volume |
| `folder_name` | 下载文件夹名称 | Download |
| `name_format` | 文件命名格式 | 发布时间 作者昵称 作品标题 |
| `image_format` | 图片格式（PNG/JPEG/WEBP/HEIC） | PNG |
| `image_download` | 是否下载图片 | true |
| `video_download` | 是否下载视频 | true |
| `download_record` | 是否启用下载记录 | true |
| `folder_mode` | 是否为每个作品创建文件夹 | false |
| `author_archive` | 是否按作者归档 | false |

## 自定义配置

如需修改配置：
1. 打开 `settings.json`
2. 修改对应参数
3. 保存文件
4. 重新运行下载

## 配置示例

### 按作者归档
```json
{
  "author_archive": true,
  "folder_mode": true
}
```
每个作者的作品会存放在独立文件夹中。

### 仅下载图片，不下载视频
```json
{
  "image_download": true,
  "video_download": false
}
```
