# My AI Agent Skills

本仓库收集了我在 Claude Code (`.claude`) 和 OpenCode (`.opencode`) 中安装和自定义的 AI Agent Skills。

## 目录结构

```
├── claude-skills/          # Claude Code 平台 Skills
│   ├── advanced-video-downloader/   # 视频下载与转录
│   ├── agent-browser/               # 浏览器自动化
│   ├── baoyu-article-illustrator/   # 文章配图生成
│   ├── baoyu-post-to-x/             # 发布到 X (Twitter)
│   ├── baoyu-url-to-markdown/       # URL 转 Markdown
│   ├── find-skills/                 # 发现和安装 Skills
│   ├── frontend-design/             # 前端设计
│   ├── instagram-downloader/        # Instagram 下载
│   ├── media-downloader/            # 智能媒体下载器
│   ├── onecontext/                  # 历史搜索
│   ├── remotion-best-practices/     # Remotion 视频制作最佳实践
│   ├── skill-creator/               # Skill 创建工具
│   ├── skill-optimizer/             # Skill 优化工具
│   ├── vercel-react-best-practices/ # React/Next.js 性能优化
│   ├── wechat-article-publisher/    # 微信公众号文章发布
│   ├── xhs-batch-downloader/        # 小红书批量下载
│   ├── xhs-comfyui-wash/            # 小红书 ComfyUI 洗图
│   ├── xhs-copywriter/              # 小红书文案优化
│   ├── xhs-downloader/              # 小红书作品下载
│   ├── xhs-image-filter/            # 小红书图片筛选
│   └── xiaohongshu-ai-workflow/     # 小红书端到端工作流
│
└── opencode-skills/        # OpenCode 平台 Skills
    ├── find-skills/                 # 发现和安装 Skills
    └── skill-creator/               # Skill 创建工具
```

## 使用方式

每个 Skill 目录下都有一个 `SKILL.md` 文件，定义了该 Skill 的功能、触发条件和使用方法。

### Claude Code
将 skill 目录复制到 `~/.claude/skills/` 即可使用。

### OpenCode
将 skill 目录复制到 `~/.opencode/skills/` 即可使用。
